# くらしの棚卸し
## サイト概要
### サイトテーマ
生活インフラを可視化・整理することを目的としたコミュニティサイトです。
​
### テーマを選んだ理由
現在、自分自身も一人暮らしをしており、家賃や光熱費、通信費、サブスクリプションなど、
毎月発生している固定費の全体像を正確に把握できていないという課題を感じています。
日々の生活の中で支払いは行っているものの、「月に合計でいくらかかっているのか」や
「どこを見直せばよいのか」が分からないまま過ごしてしまっている現状があります。

特に、一人暮らしや引っ越し、就職・転職といった生活環境の変化があった際にも、
生活インフラをまとめて整理する仕組みがなく、
結果として無駄な支出を見逃してしまうことが多いと感じました。

また、生活インフラは誰にとっても身近なテーマであるにもかかわらず、
他人がどのように管理や見直しを行っているのかを知る機会は少なく、
個人の工夫や知見が共有されにくいという問題があります。
同じような悩みを持つ人同士が情報を共有できれば、
生活コストの見直しや暮らしの質の向上につながるのではないかと考えました。

これらの背景から、コンビニエンスストアでアルバイトをしていた私は、生活インフラを「棚卸し」という形で可視化し、
自分自身の生活を振り返ると同時に、他のユーザーの工夫も参考にできる場として、
本ポートフォリオを制作しようと考えました。
### ターゲットユーザ
一人暮らしをしており、毎月かかっている固定費の全体像を把握できていない人

家賃や光熱費、通信費などの生活インフラを整理し、見直したいと考えている人

引っ越しや就職、転職などの生活環境の変化をきっかけに、支出を見直したい人
​
### 主な利用シーン
毎月の支出を振り返り、固定費を見直したい時

引っ越しや就職、転職などで生活環境が変わった時

他の人がどのように生活インフラを管理しているか知りたい時

自分の生活インフラ情報を整理・記録したい時
​
